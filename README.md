# Tic Tac Toe Game

A simple command-line implementation of the classic Tic Tac Toe game in Python using Jupyter Notebook.

## 📋 Features

- **Two-player gameplay**: Play as X and O
- **Interactive board**: Visual representation of the game board
- **Win detection**: Automatically detects when a player wins
- **Draw detection**: Identifies when the game ends in a draw
- **Input validation**: Ensures players enter valid moves
- **Error handling**: Gracefully handles invalid inputs

## 🎮 How to Play

1. The game is played on a 3x3 grid
2. Players take turns, with X going first
3. Players choose a position from 1-9 (where 1 is top-left and 9 is bottom-right)
4. The first player to get 3 of their marks in a row (horizontally, vertically, or diagonally) wins
5. If all 9 squares are filled with no winner, the game is a draw
